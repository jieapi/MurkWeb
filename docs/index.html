开放 API

将在线加密接入外部程序

MurkPHP 提供面向外部程序调用的开放 API，可用于查询加密脚本、上传 PHP 文件或 ZIP 源码包、提交加密任务、查询任务状态并下载加密结果。

API Base

10MB上传上限
ZIP结果下载

接口链路

围绕加密任务提供完整的程序化入口

开放 API 当前聚焦加密任务本身，从脚本查询到结果下载都能通过接口完成。完整路径、字段和响应结构仍以 API 文档为准。
01

查询脚本

获取可用加密脚本、版本、价格、说明和可配置选项。

02

上传文件

上传单个 PHP 文件或 ZIP 源码包，获取文件编号、哈希和价格信息。

03

提交任务

使用脚本编码、文件编号和选项提交加密任务，等待队列处理。

04

下载结果

任务成功后通过返回的下载地址获取加密后的 ZIP 结果包。

脚本列表

返回脚本编码、名称、版本、价格、插件标识和可选参数。

任务查询

支持查询任务列表、任务详情和状态选项，便于外部系统展示处理进度。

程序接入

适合构建、发布、交付系统在固定流程中自动提交加密任务。
